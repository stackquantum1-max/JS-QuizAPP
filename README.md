Quiz App

A simple, interactive Quiz Application built with HTML, CSS, and JavaScript. This project demonstrates DOM manipulation, event handling, timers, and dynamic UI updates to create an engaging quiz experience.

Features

Start and Quiz sections with smooth transitions
Multiple-choice questions with instant feedback
Timer countdown with animated progress bar
Score calculation and display
Replay and Quit functionality
Fully responsive design


Installation

Clone the repository:
git clone https://github.com/stackquantum1-max/JS-QuizAPP.git

Navigate to the project folder:
cd JS-QuizAPP
Open index.html in your browser.


Usage

Click Start Quiz to begin.
Select an answer for each question. Correct answers are highlighted.
The timer counts down for each question.
Click Next Question to proceed.
At the end, view your score.
Click Replay to start the quiz again or Quit to reload the page.
